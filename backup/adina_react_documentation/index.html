Setup Development Environment

In order to kickstart the development of your web app with Adina, the very first thing you need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Introduction

To install Adina on your local matchine, download the files from Themeforest and unzip them to the local disk on your computer.
Inside the extracted folder you will see two folders named 'adina_react_documentation' and 'adina_react'.
Open 'adina_react' with a code editor like vscode. And follow the steps below-

For npm:

Open vscode terminal
Then run : npm install
Then run : npm run dev
Now in the browser go localhost:5173

If you want to use yarn:

Open vscode terminal
Then run : yarn install
Then run : yarn dev
Now in the browser go localhost:5173

Introduction

  • public - All static files
    • favicon.png - Favicon for the website.
  • src - Content of all the react file
    • assets - All static assets like CSS, js, fonts and images
    • components - All components with relevant CSS
    • hooks - All the re-usable hooks
    • layouts - All layout components for various pages views
    • lib - All helper functions
    • pages - All page view components
    • App.jsx - Starting poing of react components
    • main.jsx - Entry poing of react app
  • eslint.config.js - Eslint configuration
  • index.html - Static index page
  • package.json - All packages have information
  • package-lock.json - All packages lock file
  • vite.config.js - Vite configuration
  • node_modules - After installing Adina, you can see this folder, this folder content all installed packages.

Fonts

By default, the template loads Inter, Manrope and Poppins font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "src/assets/css/main.css 


  @import url("https://fonts.googleapis.com/css2?family=Inter:wght@100..900&family=Manrope:wght@200..800&family=Poppins:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap");

Icons

We use Font Awesome package for icon.
To use Font Awesome, go to Font Awesome site, search your icon, select react and use that icon class.

Deploy

For deploy first run the command npm run build or yarn build
Now you will get a dist folder.
Using an FTP Client (such as Filezilla or Total Commander), you will need to upload all of the contents of the 'dist' folder to your server.
